#AI Resume Screener

I am currently working on building an NLP-based resume screening system as part of a structured plan to strengthen my fundamentals in natural language processing.
I am designing a pipeline to process resumes and job descriptions
I am planning to use spaCy and sentence-transformers for text analysis
I am implementing similarity scoring to match candidates with roles
I am building the backend using FastAPI and the frontend using React
Goal:
To develop a deployable system that demonstrates real-world NLP application and full-stack integration.
